Detailed Description

Collect data about macro expansion for string literals.

SourceRangeForStringLiterals

Instances of this class are created for each occurence of string literal expansions in macro. Each occurence allows to keep two source ranges:

  • one for the macro usage in source code (expansion usage location)
  • one for the macro definition used for expansion (macro definition location) This class also keep the original macro identifier (identifier token for the name of the macro).
